<h2>正文</h2>
<p>数据安全与AI运营公司Rubrik近日在其Rubrik FORWARD大会上宣布，推出Rubrik Agent Cloud for Anthropic‘s Claude Code and Claude Cowork。</p>
<p>该平台为企业提供了大规模部署Claude驱动型AI代理的能力，涵盖可观测性、管控力，以及行业独有的代理回滚功能，用于撤销意外操作；当破坏性操作超出版本控制范围时，还可实现不可变的代码库恢复。</p>
<p>此外，平台还增加了一层针对Claude代理的韧性保护，用于备份、监控和恢复代理行为配置。</p>
<p>当前，AI代理已能自主编写、推送和部署代码，但企业安全基础设施仍建立在假设“人始终在回路中”的基础上。</p>
<p>这种差距可能导致恶意提交、代码库勒索、提示注入以及知识产权以机器速度外泄等风险，其影响范围远超传统开发安全运维管控工具的设计能力。</p>
<p>Rubrik首席产品官Anneka Gupta表示：“企业采用Claude的速度超过了我们所见过的任何代理技术，每一位安全负责人都在问同一个问题：当代理可以自主行动时，我们如何保持控制？</p>
<p>Rubrik Agent Cloud为Claude提供了韧性层，帮助企业了解代理可以访问什么、管控其行为、回滚其操作，并在遭到破坏或篡改时恢复代码和代理配置。</p>
<p>”
Rubrik Agent Cloud for Claude Code的核心能力包括：语义AI治理引擎，业界首个旨在实时保护和管控自主代理的治理引擎，以意图驱动的治理取代静态的手动监督；代理清单，提供对所有已部署代理的风险、访问权限和策略违规情况的全面可见性；代理回滚，能够即时、精确地撤销从自定义代理到流行代理工具中开发的代理的意外操作；代码库韧性，为代码和开发者管道提供增强的回滚能力，维护GitHub和Azure DevOps仓库的连续、不可变快照；Claude代理韧性，备份、版本跟踪和恢复控制Claude代理行为的配置，并持续监控配置漂移。</p>
<p>Rubrik Agent Cloud for Claude Code现已正式可用。</p>
<p>Rubrik正将自己定位于数据保护、网络韧性和企业AI交汇处的领导者。</p>
